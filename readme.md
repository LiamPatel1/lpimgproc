# lpimgproc

- lpimgproc-core: A custom image processing library written from scratch in C++
- lpimgproc-gui: A node based frontend written with Qt6


Supported operations:
- Inverting images
- Kernel convolution

Upcoming features:
- Edge detection
- Thresholding
- Filters
- Channel manipulation
- Gamma / colour correction
- General transformations
